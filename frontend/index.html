AI Context

History Window ~100K tokens

Characters of chat history fed to Guardian per call. Keep at compaction threshold × 3.8 chars/token — or context silently drops before compaction fires. Default 380K = 100K-token compaction.

Compaction Threshold 100K tokens

Compact chat into a summary at this many tokens. Lower = fresher summaries. Higher = longer raw history.

Startup Catch-Up

Review offline security events on start. Disable to skip the daily startup report.

Daily Posture Report

Once-daily security health check. Disable if it fires too often or adds noise.

Investigate Events Trigger AI

"Investigate" severity events activate Guardian AI. Disable to raise the threshold.

Stale Agents Trigger AI

Agents with no recent heartbeat activate Guardian AI. Disable if agents go offline regularly.
